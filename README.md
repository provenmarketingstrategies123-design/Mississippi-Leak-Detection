# Mississippi Leak Detection — Website

This repository will host the landing page for Mississippi Leak Detection.

Included files:
- `index.html` — main single-page site
- `styles.css` — site styles
- `script.js` — small UI scripts

Quick preview locally:
1. Clone the repo.
2. Open `index.html` in a browser (or run a simple static server like `npx serve`).

Deploy:
- You can enable GitHub Pages from the repository settings (deploy from `main` branch).
- Alternatively, deploy to any static host (Netlify, Vercel, etc).
